Polyfuel

Gas Mileage Tracking Webapp

Created by Kristina Heredia / @polyhimnia

Ever wonder if you are actually achieving the EPA listed mpg?

This app will provide an easy way to track gas mileage for one or many vehicles

Tech Used

  • Current
    • MongoDB
    • Express
    • React
    • Node
  • Future
    • React Native
    • Edmunds API for vehicle data
